# zhhome-leyu-archive

## 项目简介

zhhome-leyu-archive 是一个用于归档和发布多个独立 HTML 页面的静态仓库。本仓库不针对任何特定域名或网站，仅作为一份通用、可浏览的页面集合，方便用户查阅和保存不同主题的内容。

## 目录结构

```
├── index.html          # 仓库首页 / 页面导航
├── pages/              # 存放独立 HTML 页面的目录
│   ├── example-page1.html
│   ├── example-page2.html
│   └── ...
└── assets/             # 资源文件（图片、样式、脚本等）
    ├── css/
    ├── js/
    └── images/
```

- **index.html**：仓库的入口页面，通常包含页面列表或导航链接。
- **pages/**：所有归档的独立 HTML 页面均放置于此目录下，每个文件为一个独立页面。
- **assets/**：存放页面所需的公共资源，如样式表、脚本文件、图片等。

## 页面归档说明

- 每个 HTML 页面均为独立、自包含的文件，可直接在浏览器中打开。
- 页面内容可能涵盖不同主题，彼此之间无直接关联。
- 归档仅为保存和展示目的，不保证内容的时效性或完整性。
- 页面中可能引用外部资源（如字体、CDN 库），但应尽可能确保离线可用性。

## 维护说明

- 本仓库不定期更新，新增页面或修正现有页面。
- 欢迎提交 Issue 或 Pull Request 以建议新增内容或报告问题。
- 若希望移除某个页面，请说明理由。
- 仓库维护者保留对内容进行筛选和调整的权利。

## 使用方式

1. 克隆或下载本仓库。
2. 在浏览器中直接打开 `index.html` 或 `pages/` 下的任意 HTML 文件。
3. 也可通过 GitHub Pages 或其他静态托管服务部署后在线访问。
